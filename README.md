# JSnake
New tab chrome extension to play snake. My favorite snake extension was removed from the Chrome Web Store, so here's my rendition. Feel free to create pull reqs

# How to use
1. Download the zip + unzip
2. From the Chrome browser: Settings &rarr; Extensions &rarr; Turn on Developer Mode &rarr; Load unpacked &rarr; Select the "game" folder within jsnake &rarr; Disable other new tab extensions
